## HR4U - Human Resource Management System

A comprehensive Human Resource Management System built to streamline workforce management, attendance tracking, payroll processing, and employee lifecycle management.

### Key Features

- **Employee Management**: Complete employee lifecycle from onboarding to exit
- **Attendance Tracking**: Real-time attendance monitoring with biometric integration
- **Leave Management**: Automated leave application and approval workflow
- **Payroll Processing**: Automated salary calculation with tax and deduction management
- **Performance Reviews**: Structured performance evaluation system
- **Reports & Analytics**: Comprehensive HR reports and dashboards

### Technologies Used

- **Backend**: PHP Laravel
- **Frontend**: Vue.js, Alpine.js
- **Database**: MySQL
- **API**: RESTful API architecture

### Outcome

Successfully deployed across multiple organizations, managing 1000+ employees with improved HR efficiency by 60%.
