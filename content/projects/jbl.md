## JBL - Jamuna Bank Limited System

A comprehensive core banking system developed for Jamuna Bank Limited, integrating various banking services and financial operations.

### Key Features

- **Core Banking Operations**: Account management, deposits, withdrawals
- **Loan Management**: Credit processing and loan tracking system
- **Inter-branch Transactions**: Seamless branch-to-branch operations
- **Compliance Module**: Regulatory compliance and reporting
- **Mobile Banking Integration**: API integration with mobile banking app

### Technologies Used

- **Backend**: PHP Laravel, Golang (microservices)
- **Frontend**: Alpine.js, Tailwind CSS
- **Database**: MySQL with replication
- **API**: RESTful and GraphQL APIs

### Outcome

Modernized banking operations serving 500,000+ customers with 99.9% uptime and improved transaction speed by 70%.
